    ``` {.sourceCode .python}

        [1, 1, scale, 1, scale, 1], dtype=data.dtype)
    ```

    to

    ``` {.sourceCode .python}

    # 6D tensors are currently not supported
    # TODO: SW-35724 revert change when they are supported

        [1 * 1, scale, 1, scale, 1], dtype=data.dtype)
    ```

    Reducing 6D tensor to 5D as 6D are not currently supported by HW.
